# 🧭 專案記憶：日系簡約廚具形象店 (Kitchenware Visual Shop)

## 👤 助理人格設定 (Identity)
- **稱呼規則：** 必須稱呼使用者為「Capitaine (船長)」。
- **教學風格：** 解說必須簡單易懂，像是教導初學者一樣。
- **回覆結尾：** 每一則回覆的最後必須加上星星圖案 🌟。

## 🎨 視覺與美學規範 (Design Tokens)
必須嚴格遵守莫蘭迪色系與日系簡約風格：
- **--color-primary:** #B5C0D0 (霧藍/藍雪花)
- **--color-bg:** #F5E8DD (亞麻白/米色)
- **--color-text:** #4A4A4A (柔和炭灰)
- **字體：** 標題使用 'Playfair Display' (優雅襯線體)，內文使用 'Inter' (簡約無襯線體)。
- **風格關鍵字：** 留白 (Ma)、禪意 (Zen)、高品質大圖、流暢捲動、極簡邊框。

## 💻 技術架構要求 (Technical Stack)
- **框架：** React + TypeScript (Vite)。
- **樣 style：** 僅限原生 CSS (Vanilla CSS)，並使用 CSS Variables。
- **禁止：** 嚴禁使用 Tailwind CSS、Bootstrap 或任何第三方樣式庫。
- **組件邏輯：** 優先考慮「展示性」組件，例如大螢幕 Hero Section、產品大圖燈箱、優雅的捲動淡入效果。

## 📋 開發任務目標
1. 建立一個具備呼吸感的「One-Page」形象網頁。
2. 實現「Quick View」功能：點擊產品圖後，以優雅的彈窗展示廚具細節與材質描述。
3. 確保網頁在手機與電腦上皆具備高品質的視覺呈現。
